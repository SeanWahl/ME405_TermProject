Detailed Description

Implements a nerf gun class to use for the term project.

An nerf gun class compatible with the Snowcinda toy machine gun. Assumes that the trigger and safety mechanisms have been altered to be shorted by two GPIO pins and external hardware. Once set up properly, activating the two pins will either arm or shoot the gun.

Initializes a nerf gun object.

Using pin objects as inputs, is able to create a nerf gun object. External hardware should be connected such that setting the armPin to high will short the flywheel's wires, and setting the shootPin to high will short the piston arm's wires.
