Detailed Description

Documents the controller class for ME 405.

Contains the "controller" class that can be used to set up controller objects that will be used in labs to do closed loop control on various devices, like positional control of a motor. This class will be general enough to do this for any device by only using only gains, setpoints, and measurements to calculate actuation signals.
